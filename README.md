Yandex-tandem
=============

Решать задачи легче в Тандеме


Сервис для создания, управления задачами.

Серверная часть построена на node.js
Инструкция по запуску проекта:

	1. Ставим nodejs 
	2. Запускаем app.js

Список необходимых модулей:
"express": "3.4.8",
"ejs": "*",
"async": "*",
"bower": "*",
"connect-mongo": "*",
"ejs-locals": "*",
"formidable": "*",
"mongoose": "*",
"nconf": "*",
"winston": "*",
"mongodb": "*"

База данных, используемая в этом проекте подключенна и в дополнительных настройках не нуждается.


Сейчас сервис находится на стадии Beta тестирования, поэтому некоторые функции недоступны (поиск, панель "дополнительно")
